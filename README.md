# eucaim_wizard_tool
The EUCAIM Wizard tool is based on the ARX Anonymization Tool (https://arx.deidentifier.org/) which is an open source software that supports privacy and risk models, methods for transforming data and methods for analyzing the usefulness of output data. ARX has been configured for the specific requirements of the EUCAIM project taking into account the imaging and clinical data characteristics, aiming at balancing usability and security aspects.
